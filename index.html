Hướng dẫn tích hợp Google Sheets & API thực tế

Bạn đang chạy chế độ giả lập (Demo Mode). Để kết nối cơ sở dữ liệu thật của riêng bạn, vui lòng thực hiện các bước sau:

  1. Tạo 1 Google Spreadsheet chứa 4 sheet con đặt tên chính xác là: Users, Matches, Predictions, Leaderboard_Cache.
  2. Tạo Google Cloud Project, bật Google Sheets API v4 và cấu hình OAuth Client ID (loại Web Application).
  3. Mở file js/config.js và điền các tham số: SPREADSHEET_ID, GOOGLE_CLIENT_ID.
  4. (Tùy chọn) Điền token từ football-data.org vào FOOTBALL_API_TOKEN để lấy lịch thi đấu thật.
